## 名前

fadee

## 概要(132 文字以内)

視聴済みの動画を YouTube 上で非表示にして、未視聴の動画を見つけやすくする拡張機能です。

## 詳細な説明

fadee は YouTube 上で視聴済みの動画を非表示にして、未視聴の動画をひと目で見つけやすくします。

■ 主な機能

- YouTube が視聴済みと判定した動画を自動で非表示(「視聴済みとみなす再生率」のしきい値を調整可能)。
- 任意の動画カードに付くワンクリックの「視聴済みにする」ボタンで手動コントロール。
- スコープ別トグル: チャンネル · 動画 / ショート / ライブ、登録チャンネル、ホーム、検索結果。
- 追加オプション: Shorts を全部隠す、ホームのテーマ別シェルフを隠す、おすすめにフィルターを適用しない。
- 視聴済み ID は chrome.storage.sync に保存され、Chrome 自体の同期機能を通じてご自身の Chrome プロファイル間で同期します。fadee 自身は外部に送信しません。
- ツールバーアイコンは OS のダーク / ライトモードに追従。
- ポップアップで現在の sync ストレージ使用量を確認、手動マーク済みリストのクリアも可能。

■ 使い方

1. YouTube を開き、対象スコープのページ(チャンネル動画、登録チャンネル、ホーム、検索)を表示。
2. YouTube が視聴済みと判定した動画は自動で非表示になります。
3. 手動で非表示にしたい場合は、動画カードに表示される「視聴済みにする」ボタンをクリック。
4. ツールバーのポップアップから、マスタートグル、スコープ切り替え、視聴済みしきい値の調整、手動マーク済みリストのクリアが行えます。

■ スコープ

- 動作対象は https://www.youtube.com/* のみ。
- 保存するのは拡張機能のローカル設定と手動マーク済み動画 ID リストのみ(chrome.storage 経由)。
- 拡張機能自身はネットワーク通信を行いません。
- オープンソース: https://github.com/ichi0g0y/fadee

## 単一目的の説明

fadee は、YouTube 上で既に視聴済みとなった動画(YouTube の再生進捗から自動判定されたもの、またはユーザが手動でマークしたもの)を非表示にして、未視聴の動画を見つけやすくします。

## 権限の正当化

- **storage** — 手動マーク済み動画 ID リストと、いくつかの UI 設定(マスタートグル、スコープ別トグル、視聴済みしきい値)を `chrome.storage.sync` に永続化するために必要です。これがないと、リロード時にリストや設定が失われます。
- **offscreen** — `matchMedia("(prefers-color-scheme: dark)")` で OS のカラースキーム設定を読み取り、結果を Service Worker に伝えるための小さな offscreen ドキュメントをホストします。MV3 の Service Worker 文脈から `matchMedia` を観測する唯一の公認手段で、ツールバーアイコンをライト / ダーク変種で切り替えるためだけに使用します。
- **host_permissions: https://www.youtube.com/\*** — YouTube ページの DOM ノードを読んで動画カードを特定し、「視聴済みにする」ボタンの注入と、視聴済みカードへの非表示 CSS の適用のために必要です。

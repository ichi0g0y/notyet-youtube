# fadee

`fadee` is a Chrome extension that hides watched videos on YouTube channel pages, so unwatched ones stand out.

`fadee` は YouTube のチャンネルページで、視聴済みの動画をフェードさせて、未視聴の動画を目立たせる Chrome 拡張機能です。
